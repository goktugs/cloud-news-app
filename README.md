# Personalized News App



Dtcloud fullstack developer homework. It gives you various news and you can singup and make them personalized

## Installation

Project is kinda monorepo so you need to run Backend and Frontend too.

# For Backend

```bash
After clone the repo
```
```bash
cd server
```

```bash
code . then npm install
```

```bash
npm run dev
```

# For Frontend


```bash
After clone the repo
```
```bash
cd client
```

```bash
code . then type yarn 
```

```bash
yarn dev
```
